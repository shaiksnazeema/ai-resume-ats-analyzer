
🤖 AI Resume ATS Analyzer

Analyze how well your resume performs against Applicant Tracking Systems (ATS) and job descriptions using AI.

 🚀 Overview

AI Resume ATS Analyzer is a web-based application that evaluates resumes by comparing them with job descriptions. It uses Natural Language Processing (NLP) techniques to calculate ATS compatibility, keyword matching, and provide actionable feedback.

 ✨ Features

* 📄 Upload and analyze resumes
* 🎯 ATS compatibility scoring
* 🔍 Keyword matching with job descriptions
* 📊 Match percentage calculation
* 💡 Suggestions to improve resume quality
* 🌐 Simple web interface

🛠️ Tech Stack
*Backend: Python (Flask)
 *Frontend:HTML, CSS
 *Libraries: NLP-based text processing (e.g., NLTK / spaCy if used)

 ⚙️ Installation & Setup

1. Clone the repository:
git clone https://github.com/shaiksnazeema/ai-resume-ats-analyzer.git

2. Navigate to the project folder:
cd ai-resume-ats-analyzer
3. Install dependencies:
pip install -r requirements.txt
4. Run the application:
python app.py
5. Open in browser:
http://127.0.0.1:5000/

 📊 How It Works

1. Upload your resume
2. Paste a job description
3. The system analyzes both
4. Get:
 * ATS score
 * Keyword match percentage
 * Missing keywords
 * Improvement suggestions
